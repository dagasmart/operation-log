## BizAdmin 操作日志

## 效果

增加操作日志记录

## 安装

#### zip 下载地址

[https://github.com/DagaSmart/operation-log/archive/refs/heads/master.zip](https://github.com/DagaSmart/operation-log/archive/refs/heads/master.zip)

#### composer

```bash
composer require dagasmart/operation-log
```

## 使用说明

1. 安装扩展
2. 在扩展管理中启用扩展
